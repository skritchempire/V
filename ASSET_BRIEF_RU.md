# ТЗ на генерацию изображений для Shelter11

Этот файл нужен, чтобы ты мог просто скопировать описания в ChatGPT/генератор изображений и получить полный набор ассетов.

## 1) Сколько нужно изображений

### Базовый обязательный набор
- **Комнаты:** 15 шт.
- **Персонажи:** 10 шт.
- **Итого минимум:** **25 изображений**.

### Рекомендуемый расширенный набор
- Компаньоны: 2 шт. (пёс, дрон)
- Варианты внешки (например, 2–3 версии на каждого жителя): +20–30 шт.
- **Итого комфортно:** 45–60 изображений.

## 5) Единые параметры генерации (чтобы всё выглядело цельно)

Используй одинаковый стиль для всех промптов:

- **Стиль:** grimdark post-apocalypse bunker, semi-realistic game art
- **Ракурс комнат:** side view / isometric-like side scene
- **Ракурс персонажей:** full body sprite, standing pose, transparent background
- **Свет:** low-key, cinematic, volumetric dust
- **Без текста/логотипов/водяных знаков**

Рекомендуемые размеры:

- Комнаты: `1536x1024` или `1280x720`
- Персонажи: `768x1024` (потом при необходимости ресайз)

## 6) Шаблон промпта для ChatGPT

### Для комнаты

```text
Create a game background for a Telegram mini app.
Style: grimdark post-apocalypse underground bunker, semi-realistic, cinematic lighting, no text, no watermark.
Room type: <ROOM_NAME>.
Details: <ROOM_DESCRIPTION_FROM_TABLE>.
Composition: side-view room interior, readable silhouettes, space for UI overlays.
Output: webp-friendly, high detail.
```

### Для персонажа

```text
Create a full-body game character sprite for a Telegram mini app.
Style: grimdark post-apocalypse bunker, semi-realistic, consistent with other characters.
Character type: <CHAR_NAME>.
Details: <CHAR_DESCRIPTION_FROM_TABLE>.
Pose: neutral standing pose, front/3-4 view.
Background: transparent or plain chroma for easy cutout.
No text, no watermark.
```
